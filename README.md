# 🚖 Ola Ride Booking Analysis Dashboard

## 📌 Project Overview
The **Ola Ride Booking Analysis Dashboard** is an end-to-end Data Analytics project designed to analyze ride booking data and generate actionable business insights. This project demonstrates the complete analytics workflow, starting from data cleaning and preprocessing in Excel, performing data analysis using MySQL, and building an interactive dashboard in Power BI.
The primary objective of this project is to understand booking patterns, revenue generation, cancellation trends, customer behavior, and vehicle performance. By transforming raw booking data into meaningful visualizations and KPIs, the dashboard helps stakeholders make data-driven decisions and improve operational efficiency.
The dataset consists of over **100,000 ride booking records**, which were cleaned, transformed, and analyzed to uncover key trends and performance metrics. SQL queries were used to extract valuable insights, while Power BI was utilized to create dynamic reports and interactive visualizations for better business understanding.
